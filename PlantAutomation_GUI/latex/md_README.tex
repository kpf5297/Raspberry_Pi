The Plant Care System GUI is a graphical user interface application developed for managing and monitoring a plant care system. It provides an intuitive interface for controlling lights, water pumps, and monitoring soil moisture levels.\hypertarget{md_README_autotoc_md2}{}\doxysection{Features}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Automated Control\+:}} The system includes automated control of lights and water pumps based on specified schedules.
\item {\bfseries{Manual Control\+:}} Users can manually control lights and water pumps through the graphical interface.
\item {\bfseries{Soil Moisture Monitoring\+:}} The GUI displays real-\/time soil moisture levels from soil sensors in different plant shelves.
\item {\bfseries{Logging\+:}} Events and changes made to the system are logged for reference.
